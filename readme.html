匿名薇閣自動軟體使用說明書

使用說明

  1. 打開main.exe，會先出現一個黑色框框，這只有在登入時會用到，不太需要管；接下來會有一個Chrome視窗，盡量不要去碰。
  2. 如果你有登入過本軟體，稍帶一會就可以開始審核。若沒有，請在黑色視窗內輸入您的帳號密碼。
  3. 登入成功後，會出現另一個視窗，各部件敘述如下:
    • 頂部的數字框:輸入貼文的編號
    • 中間的輸入框:輸入貼文的內文
    • 核准並送出:核准目前審核中的貼文並將其製成圖片
    • 核准:核准目前審核中的貼文(不製圖)
    • 駁回:駁回目前審核中的貼文
    • 送出:將目前審核中的貼文製成圖片
    • 自動換行:整理內文版面(測試中)
    • 刷新:重新整理網頁
    • 略過:跳過這則貼文
    • 退出:退出軟體
  4. 視窗產生後，第一則貼文內容會顯示在輸入框
  5. 修改貼文的編號
  6. 點自動換行，再視情況手動修改
  7. 送出時建議是先單獨送出，圖片沒問題再核准
  8. 跳出"成功"的視窗之後，可以再"output"資料夾內找到圖片
  9. 打開ig建立貼文，將圖片上傳，打完回覆後空一行，Ctrl+V 貼上Hashtag。
  10. 審核下一篇時，貼文編號會自動修改。

注意事項

  • 電腦必須安裝 Edge ，建議更新到最新版本。
  • 若審核途中動到Edge視窗的匿薇網頁，請點擊"刷新"，以免發生錯誤
  • 審核途中若有送出後主視窗貼文未更新的狀況，同上，若無效，請在退出軟體後重新打開
